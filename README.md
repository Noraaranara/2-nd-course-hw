✨ Описание проекта: "Website with mini games":
Цель проекта — разработать интерактивный веб-сайт с коллекцией увлекательных мини-игр, доступных прямо в браузере. Сайт должен быть простым в использовании, с ярким дизайном и адаптивной вёрсткой для игры на любых устройствах.

✨ Запуск проекта:
1. Клонируйте репозиторий:
   git clone https://github.com/имя/проект.git

2. Перейдите в папку проекта
    cd проект

3. Откройте файл index.html в браузере.

✨ Использованные технологии:
HTML5, CSS (SASS), JavaScript, БЭМ, Git
